# 🌿 ResearchMindAI

An AI-powered research paper assistant built using FastAPI, React, ChromaDB and Firebase Authentication.

## Features
- PDF Upload
- Semantic Search using RAG
- AI Question Answering
- Summary Generation
- Source Citations
- Google Sign-In
- Chroma Vector Database

## Tech Stack
Frontend:
- React
- Vite
- Firebase Auth

Backend:
- FastAPI
- ChromaDB
- Sentence Transformers
- Groq API
